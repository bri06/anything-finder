# anything-finder

## Buscador de peliculas básico con poylmer 2

### Pasos a realizar:
Tienes que tener instalado lo siguiente:
```shell
npm install -g polymer-cli
```

```
npm install -g bower
```

Para instalar las dependencias:
```
bower install
```

Para lanzar la aplicación:
```shell
polymer serve -o
```
